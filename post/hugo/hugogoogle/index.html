在Hugo中使用Google Analytics

使用环境#

开发环境: Windows 11
Hugo版本: 0.121.2
Git版本: 2.38.1
Theme: PaperMod

前提#

首先在google analytics 注册账号并添加自己的域名

设置GoogleAnalytics参数#

在config.yaml中配置如下参数，其中G-MEASUREMENT_ID在数据流详情页

services:
  googleAnalytics:
    ID: G-MEASUREMENT_ID

新建Google Analytics模板#

在layouts\_internal\目录下新建google_analytics.html(目录若不存在,自行创建)

<!-- Google tag (gtag.js) -->
<script async src="https://www.googletagmanager.com/gtag/js?id=自己的G-MEASUREMENT_ID"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());

  gtag('config', '自己的G-MEASUREMENT_ID');
</script>

引用Google Analytics模板#

在themes\PaperMod\layouts\partials目录下，打开head.html，添加如下内容:

<head>
{{- if not .Site.IsServer }}
	{{ template "_internal/google_analytics.html" . }}
{{- end }}
</head>
